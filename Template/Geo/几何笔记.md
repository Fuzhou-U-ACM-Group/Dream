#  几何笔记

## 球面坐标

- 地球上经纬度知识
  - 经度：本初子午线向东为东经，西为西经，西经为负，东经为正，-180° ~ 180°
  - 维度：赤道向北为北纬，赤道向南为南纬，南纬为负，北纬为正，-90° ~ 90°
  - 经纬度：1度 = 60分，1分 = 60秒
- 经纬度坐标与三维坐标相互转化
  - $x = cos(w) * cos(j) $
  - $y = cos(w) * sin(j)$
  - $z = sin(w)$
  - $w = acos(z)$
  - $j = asin(\frac{y} { \sqrt{1 - z ^2}}) $
- 两点的球面距离 = 两点所在大圆优弧对应圆心角 * 球半径
  - $ = acos(cos(wA) * cos(wB) * cos(jB - jA) + sin(wA) * sin(wB)) * R$
  - $ = acos(dot(A, B) / R / R) * R$
